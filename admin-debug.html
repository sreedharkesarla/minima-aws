Minima Admin Debug Page

Instructions:

  1. Open a new tab: http://localhost:3001
  2. Before logging in, press F12 to open Developer Tools
  3. Go to the Console tab
  4. Login with: admin / Admin@123
  5. Watch for any RED error messages in console
  6. Report back what you see

What to look for in Console:

  • If you see: "Failed to load module" → JS file loading issue
  • If you see: "Cannot read property" → React component error
  • If you see: "Unexpected token" → JavaScript syntax error
  • If you see: Nothing (blank console) → Check Network tab for failed requests

Network Tab Check:

  1. In Developer Tools (F12), click "Network" tab
  2. Refresh the page (Ctrl+R)
  3. Look for any RED files (failed to load)
  4. Click on the failed file to see the error
