🚀 Minima Chat Widget - Embedding Demo

Note: This demo shows three integration methods for the Minima chat widget. Make sure the chat backend service is running on port 8003.

1. Web Component (Custom Element)

<script type="module" src="minima-chat.js"></script> <minima-chat data-endpoint="ws://localhost:8003/chat/demo/demo-001/all" data-conversation-id="demo-001" style="--minima-chat-accent: #4f46e5;" ></minima-chat>

2. iframe Embed (Maximum Isolation)

<iframe src="https://chat.minima.ai/embed?conv=demo-002" title="Minima Chat" sandbox="allow-scripts allow-same-origin" style="width: 100%; height: 500px; border: 0;" ></iframe>

3. JavaScript SDK (Programmatic Control)

import { MinimaChatWidget } from '@minima/chat-widget'; // Create widget instance const chatWidget = document.createElement('minima-chat'); chatWidget.dataset.endpoint = 'ws://localhost:8003/chat/test/demo-003/all'; chatWidget.dataset.conversationId = 'demo-003'; chatWidget.style.cssText = '--minima-chat-accent: #4f46e5; height: 500px;'; // Mount to container document.getElementById('chat-container').appendChild(chatWidget); // Listen to events (custom) chatWidget.addEventListener('minima-chat:message', (e) => { console.log('New message:', e.detail); }); chatWidget.addEventListener('minima-chat:connect', () => { console.log('Connected!'); });

Integration Features

  • ✅ Framework-agnostic - Works with React, Vue, Angular, or vanilla JS
  • ✅ Shadow DOM - Encapsulated styles prevent conflicts
  • ✅ WCAG 2.2 compliant - ARIA role="log", live regions, keyboard navigation
  • ✅ Auto-reconnection - Handles network interruptions gracefully
  • ✅ Theming - CSS custom properties for accent color and font
  • ✅ Streaming responses - Real-time message display
  • ✅ Security - iframe sandbox support for maximum isolation
